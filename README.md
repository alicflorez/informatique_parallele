## Install du bordel

Télécharger et installer MPI

```
$ wget https://www.open-mpi.org/software/ompi/v2.0/downloads/openmpi-2.0.1.tar.gz
$ tar -xvzf openmpi-2.0.1.tar.gz 
$ cd openmpi-2.0.1/
$ ./configure 
$ make all
# make install
```


## Lancer le bordel

```
$ make makefile all
```
